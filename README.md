# Perx Test Service

Тестовое FastAPI-приложение с core-модулем.

---

В рамках задания реализован `core`-модуль, который отвечает за:

* конфигурирование приложения через переменные окружения
* управление подключением к PostgreSQL
* управление подключением к Redis
* интеграцию инфраструктурных ресурсов в FastAPI через Dependency Injection
* управление жизненным циклом приложения через `lifespan`
* базовые health-check endpoints

## Обоснование принятых архитектурных решений

* Централизация инфраструктуры
* Слабая связанность
* Простота расширения
* Управляемый lifecycle
* Fail-fast поведение
